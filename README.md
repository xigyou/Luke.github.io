# 为什么要知识管理(Why?)

从大学开始我就开始使用OneNote来进行知识管理，但是以前主要是进行资料存放，记笔记。没有真正形成体系。虽然进行了一些分类整理，但是总是遇到不知道该如何分类，已经经常出现同一方面的内容放到多处笔记分区和页的情况。
直到最近尝试按照书本的方式建立知识体系的时候在知乎上找到了文韬的这篇文章，顿觉醍醐灌顶，他做的非常漂亮，系统。大家可以参考。
